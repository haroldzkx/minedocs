# 【海外 Apple ID 注册教程】

前提条件

- 可以正常接收短信的国内手机号码

- 可以正常接收邮件的电子邮箱

- 注意: 若该电子邮箱已经注册过 Apple ID，请更换为未注册过 Apple ID 的电子邮箱账号

# 1.打开 appleid 首页

在 Safari 浏览器内打开苹果官网注册网址：https://appleid.apple.com/account

![](./images/apple.1.png)

然后点击继续

![](./images/apple.2.png)

# 2.验证邮箱

填写完所有的信息点击继续之后，这时会收到一封验证 Apple ID 的电子邮件，需要把邮箱验证码填写上去

![](./images/apple.3.png)

完成邮箱验证之后，系统会再发送一条短信给你验证手机号，需要把手机号验证码填写上去

![](./images/apple.4.png)

# 3.修改国家或地区

完成手机号码验证后，会出现账号信息，电脑端点击左侧的个人信息，再点击右侧的国家与地区（手机端操作请继续往下看）

![](./images/apple.5.png)

完成手机号码验证后，会出现账号信息，手机端，点击三角按钮，接着点击个人信息，再点击国家与地区

![](./images/apple.6.png)

![](./images/apple.7.png)

![](./images/apple.8.png)

点击更改国家与地区

![](./images/apple.9.png)

更改国家为美国，是用浏览器打开美国人虚构信息生成的网址：http://www.haoweichi.com/，

根据信息，填写对应内容。省/市的选择，根据美国洲全称简称一览表：http://www.library.sh.cn/tsgc/lxzn/留学指南(美).files/zhou.table.htm

找出对应的省/市，填写完成后点击更新，更新地区信息。

![](./images/apple.10.png)

![](./images/apple.11.png)

![](./images/apple.12.png)

# 4.退出 App Store 账号

1、进入 App Store，点击右上方的头像，进入到账户页面

![](./images/apple.13.png)

2、滑动该页面至底部，点击“退出登录”即可退出商店 ID，再点击”完成”

![](./images/apple.14.png)

# 5.在 App Store 登录刚注册的 Apple ID

1、再次点击右上角的头像，进入账号登录页面

![](./images/apple.15.png)

2、输入账号、密码，点击“登录”，进行登录

![](./images/apple.16.png)

3、输入验证码，成功检验后，即登录成功 Apple ID

![](./images/apple.17.png)

4、若出现以下提示信息，则按步骤进行检查操作，完成检查

![](./images/apple.18.png)

5、点击“同意条款与条件”的按钮，再点击”下一页”

![](./images/apple.19.png)

6、下图请不要填写任何信息，直接点击”下一页”

![](./images/apple.20.png)

7、点击“继续”，即可进行下载

![](./images/apple.21.png)

# 6.在 App Store 搜索软件进行下载

例如 QuickQ，WhatsApp 等软件
