# 英语学习误区

- 误区 1: 把刻苦等同于有效（要想英语学好，光刻苦不行，还得找方法）

- 误区 2: 盲从（学口语多开口不一定有效）

- 误区 3: 违背自然规律（学语言的自然规律是：先听说，后读写）

- 误区 4: 出国就一定能学好英语（国内其他地区的方言都学不会，更不用说英语了）

- 误区 5: 学英语就是枯燥乏味（要自己发觉学英语中的乐趣）

# 英语学习常见问题答疑

- 多久能学好英语：要持之以恒地学，学以致用，经常用就不会忘

- 口音：学口音是很快可以看见效果的，好的口音给人自信，美音，英音随便选一个学

- 英语多好才算学得够好：够用就行（通过考试 → 精通专业英语 → 能够融入国外生活）

- 第二外语：有需要才学，没需要就不要浪费时间

- 自学英语的难题：对的方法，坚持学习

英语当中的学习分为三大阶段。

- 基础英语学习阶段：语法（读懂长句子）、词汇、口音
- 中级英语学习阶段：听、说、读、写
- 高级英语学习阶段：口译，笔译
