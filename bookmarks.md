<summary>备份微信聊天记录</summary>

Mac 电脑上的备份位置

```bash
/Library/Containers/com.tencent.xinWeChat/Data/Library/Application\ Support/com.tencent.xinWeChat/2.0b4.0.9
```

在 Mac 电脑系统内，微信的备份文件会存放在 2.0b4.0.9 的 Backup 文件夹内，并且备份目录的路径不会改变。每一个微信账号的聊天记录备份对应此目录下的一个文件夹，名称通常为一长串字母和数字的组合。

备份后打包文件

```bash
tar -czvf 20201110.20240501.tar.gz 16035ef422c8f9e6892ae1653e7afba0
```

然后上传 20201110.20240501.tar.gz 到网盘
就可以删除 Backup 文件夹了

解压打包文件

```bash
tar -xzvf 20201110.20240501.tar.gz
```

<details>
<summary>海外 Apple ID 注册教程</summary>

前提条件

- 可以正常接收短信的国内手机号码

- 可以正常接收邮件的电子邮箱

- 注意: 若该电子邮箱已经注册过 Apple ID，请更换为未注册过 Apple ID 的电子邮箱账号

<details>
<summary>1.打开 appleid 首页</summary>

在 Safari 浏览器内打开苹果官网注册网址：https://appleid.apple.com/account

- 国家或地区先选择中国大陆

- 出生日期要大于 18 岁

- 新的 Apple ID 要未注册过的邮箱账号

填写好信息后，点击继续

</details>

<details>
<summary>2.验证邮箱</summary>

- 填写完所有的信息点击继续之后，这时会收到一封验证 Apple ID 的电子邮件，需要把邮箱验证码填写上去

- 完成邮箱验证之后，系统会再发送一条短信给你验证手机号，需要把手机号验证码填写上去

</details>

<details>
<summary>3.修改国家或地区</summary>

完成手机号码验证后，会出现账号信息，电脑端点击左侧的【个人信息】，再点击右侧的【国家或地区】（手机端操作请继续往下看）

完成手机号码验证后，会出现账号信息，手机端，点击【三角按钮】，接着点击【个人信息】，再点击【国家与地区】

更改国家为美国，是用浏览器打开美国人虚构信息生成的网址：http://www.haoweichi.com/，

根据信息，填写对应内容。省/市的选择，根据美国洲全称简称一览表：http://www.library.sh.cn/tsgc/lxzn/留学指南(美).files/zhou.table.htm

找出对应的省/市，填写完成后点击更新，更新地区信息。

- 国家或地区：选择为美国

- 付款方式：选择为无，或者不填写

- 账单寄送地址：填写前面 haoweichi 网站里的地址信息

- 楼号、单元号、房间号：可不填

- 城市：填写 haoweichi 网站里的

- 省/市：根据州的英文简称，找出对应的州名全称

- 邮编：填写 haoweichi 网站里的

- 电话号码：填写 haoweichi 网站里的

最后点击【更新】

</details>

<details>
<summary>4.退出 App Store 账号</summary>

1、进入 App Store，点击右上方的头像，进入到账户页面

2、滑动该页面至底部，点击“退出登录”即可退出商店 ID，再点击”完成”

</details>

<details>
<summary>5.在 App Store 登录刚注册的 Apple ID</summary>

1、再次点击右上角的头像，进入账号登录页面

2、输入账号、密码，点击“登录”，进行登录

3、输入验证码，成功检验后，即登录成功 Apple ID

4、若出现以下提示信息，则按步骤进行检查操作，完成检查

> 此 Apple ID 尚未在 iTunes 商店使用过。轻点“检查”以登录，然后检查您的账户信息。

5、在【完成创建 Apple ID】页面中先点击【同意条款与条件】按钮，再点击【下一页】

6、在【完成创建 Apple ID】页面中，如果出现要求填写【付款方式，账单接受人姓名，账单寄送地址】等信息，**不要填写任何信息**，直接点击【下一页】

7、点击“继续”，即可去下载 App 了

</details>
